Estructura de imagenes del inicio

1. Carpeta publicada en el home:
img/demo/home-assets/

Aqui van las imagenes que YA usa index.php en las secciones nuevas del inicio:
- menu principal personalizado
- banner principal
- banners promocionales

2. Carpeta de originales/base:
img/demo/home-assets-originals/

Usa esta carpeta para guardar:
- imagenes originales descargadas
- recortes base
- versiones editables antes de exportar

Forma recomendada:
- primero guarda el original en home-assets-originals/apple/
- cuando ya este listo para publicar, copialo a home-assets/apple/
- si cambias el nombre del archivo final, actualiza la ruta en index.php
